## Quick Setup

run `mvn clean install`

## Manual Setup

1. run in terminal `mvn clean`.
2. run in terminal `mvn compile`.

## Run Tests

To run all tests  `mvn test`.

To run individual methods tests `mvn test -Dtest="testname#methodname"`
